Swiftly: AI Powered To-do List

Revolutionize your productivity with our AI-driven to-do list app.

Project Description

Swiftly is an AI-powered productivity dashboard that aggregates, curates, and displays information about the user’s schedule to remove the stress of time management. Our app employs a machine-learning algorithm that helps make decisions for the users on when to complete certain tasks, freeing up time and allowing them to complete their tasks more efficiently.

Our Project

Home Screen

This is the main page of our app, which gives an overview of the user’s day. It includes a list of all the tasks the user has completed, and still needs to get done, as well as the allotted times to complete those tasks.

Settings View

This screen allows the user to add and edit recurring events, or schedules, that act as a template for the app to build around when adding new tasks. It also allows the user to select different themes, giving the app a slightly more personalized feel.

Week View

This screen allows the user to view their week at a glance, allowing the user to see which tasks they have to complete on which day and allows them to select specific days to bring up all the information for that given day.

Month View

This screen displays the month view for the user. It displays a heatmap to see which days have the most tasks to complete on that day. Selecting a day bring up detailed information about that day to the user where they can view tasks and times to complete them.
